## Oliver's dotfiles

### Installation

*In ~/.profile:*

    source ~/.dotfiles/sh/common.sh
    
    source ~/.dotfiles/sh/darwin.sh
    or
    source ~/.dotfiles/sh/linux.sh

*In ~/.zshrc:*

    source ~/.dotfiles/sh/zsh/config.zsh

    source ~/.dotfiles/sh/zsh/prompt-min.zsh
    or
    source ~/.dotfiles/sh/zsh/prompt.zsh
